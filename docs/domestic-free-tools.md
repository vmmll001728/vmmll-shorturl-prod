## 2. 容器镜像加速器

### 2.1 阿里云容器镜像加速器 ⭐ 推荐

**官网**: https://cr.console.aliyun.com

**免费额度**:
- 个人专属加速器地址（每个账号独立）
- 拉取速度：通常 5-10 MB/s
- 适用场景：阿里云服务器 + 本地开发

**实名认证**: ✅ 需要

**配置步骤**:

1. 登录阿里云控制台
2. 进入「容器镜像服务」→「镜像工具」→「镜像加速器」
3. 获取专属加速地址（格式：`https://xxxx.mirror.aliyuncs.com`）
4. 配置 Docker daemon.json：

```json
{
  "registry-mirrors": ["https://xxxx.mirror.aliyuncs.com"]
}
```

5. 重启 Docker 服务：
```bash
sudo systemctl daemon-reload
sudo systemctl restart docker
```

## 6. 域名 & SSL 证书

### 6.1 腾讯云 SSL 证书 ⭐ 推荐

**官网**: https://console.cloud.tencent.com/ssl

**免费额度**:
- 证书类型：DV（域名型）
- 有效期：3 个月（2024 年 3 月后改为 3 个月，原为 1 年）
- 证书数量：每个账号最多 20 个
- 支持域名：单域名（不支持通配符）

**实名认证**: ✅ 需要

**申请步骤**:
1. 登录腾讯云 SSL 证书控制台
2. 点击「申请免费证书」
3. 填写域名信息
4. 选择验证方式（自动 DNS 验证 / 手动 DNS 验证 / 文件验证）
5. 完成域名验证
6. 下载证书并部署到服务器

### 6.4 Let's Encrypt（国际）

**官网**: https://letsencrypt.org

**免费额度**:
- 证书类型：DV
- 有效期：90 天（可自动续期）
- 支持通配符证书（DNS 验证）
- 数量：无限制

**实名认证**: ❌ 不需要

**申请工具**:
- Certbot（官方推荐）
- acme.sh（轻量级 Shell 脚本）

**配置步骤**（使用 Certbot）:
```bash
# 安装 Certbot
sudo apt-get install certbot python3-certbot-nginx

# 申请证书（Nginx 示例）
sudo certbot --nginx -d example.com -d www.example.com

# 自动续期
sudo certbot renew --dry-run
```

## 📌 重要提示

### 关于实名认证
国内平台几乎都要求实名认证（个人身份证或企业营业执照）。这是为了符合中国法律法规（网络安全法、数据安全法等）。

### 关于免费额度的变化
云服务商的免费额度可能会随时调整（如腾讯云 SSL 证书从 1 年改为 3 个月）。建议在使用前访问官网确认最新政策。

### 推荐配置组合
对于国内小型团队，推荐以下组合：
1. **代码托管**：Gitee（开源项目）+ CODING（私有项目）
2. **容器镜像**：阿里云加速器（主力）+ 网易云加速器（备用）
3. **CI/CD**：CODING CI 或 Gitee Go
4. **云数据库**：阿里云 RDS Serverless（开发测试）
5. **云存储**：腾讯云 COS（50GB 免费额度足够个人使用）
6. **SSL 证书**：Let's Encrypt（自动化）或 JoySSL（长期免费）
7. **监控日志**：阿里云 Prometheus 或腾讯云 TMP
